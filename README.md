# bprotocol-deploy

## Requirements

For `testchain`, launch the testchain before.

For `testnet`, set the following environment variables

    export ETH_FROM=0x16Fb96a5fa0427Af0C8F7cF1eB4870231c8154B6

    export ETH_RPC_URL=127.0.0.1:2000

### Start deployment

    . ./deploy.sh <network>

Supported network

    testchain
